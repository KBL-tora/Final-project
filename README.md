# Macon County Website Redesign
A responsive, accessible, multi‑page website built using Bootstrap 5 and a customized version of the Modern Business layout.  
Created for WEB‑140 to demonstrate Bootstrap integration, accessibility, responsive design, and multi‑page site structure.

## 🌿 Project Overview
This project redesigns the Macon County, NC website using:
- Bootstrap 5 (CDN)
- Custom nature‑inspired color palette
- Split‑hero homepage layout
- Circular county seal (SVG)
- Two‑level dropdown navigation
- 7 fully structured pages

## 🧭 Pages Included
- **Home** – Split hero, quick access cards  
- **Residents** – Utilities, schools, transportation  
- **Government** – Commissioners, meetings & agendas  
- **Departments** – Health, Public Safety, Public Works  
- **Business** – Permits, zoning, economic development  
- **News** – Announcements and updates  
- **Contact** – Contact form  

## 🎨 Design Choices
- Nature‑inspired palette (greens + earth tones)
- Circular county seal with mountain icon
- Modern civic building hero image
- Clean, accessible typography

## ♿ Accessibility
This project follows WCAG 2.1 AA guidelines:
- Semantic HTML  
- Proper alt text  
- High‑contrast color palette  
- Keyboard‑friendly navigation  
- WAVE‑tested pages  

## 🚀 Deployment
This site is deployed using GitHub Pages and WCET server hosting.
